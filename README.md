# DGraph Quickstart

## Installation

Install dgraph:

```
curl https://get.dgraph.io > /tmp/get.sh
less /tmp/get.sh
sh /tmp/get.sh
```

Install Flask and other Python dependencies:

```
mkvirtualenv -p python3 dgraph
pip install -r requirements.txt
```
